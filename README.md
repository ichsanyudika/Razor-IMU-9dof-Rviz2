## Build & Run
    
  1. Connect the IMU via USB.
    
  2. Build:

    git clone https://github.com/ichsanyudika/Razor-IMU-9dof-Rviz2.git
    
    cd ~/Razor-IMU-9dof-Rviz2
    
    colcon build
    
    source install/setup.bash
    
  3. Launch:
    
    ros2 launch imu_serial serial_imu_node.py

    rviz2
